class streamsx.topology.schema.CommonSchema¶

Bases: enum.Enum

Common stream schemas for interoperability within Streams applications.

Python - Stream constains Python objects Json - Stream contains JSON objects. Streams with schema Json can be published and subscribed between Streams applications implemented in different languages. String - Stream contains strings. Streams with schema String can be published and subscribed between Streams applications implemented in different languages. Binary - Stream contains binary tuples. NOT YET SUPPORTED IN Python.
